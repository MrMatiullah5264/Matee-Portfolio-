# Personal Portfolio Website

## Project Description
This project is a personal portfolio website developed to showcase my profile as a Software Engineering student.  
The website highlights my skills, projects, and contact information in a clean and professional layout.  
It serves as an online presence for recruiters and collaborators to understand my technical background and work.
The portfolio was designed and developed using modern front-end technologies and focuses on simplicity, responsiveness, and usability.

## Technologies Used
- HTML5  
- CSS3  
- JavaScript  
- Visual Studio Code  

## Features
- Responsive and clean user interface  
- About Me section introducing personal and academic background  
- Projects section to showcase completed work  
- Skills section highlighting technical expertise  
- Contact section for easy communication  
- Integrated links to GitHub and LinkedIn profiles
